About

This file is a test file for markdown to examine how pandocs works. Pandocs converts simple text file into PDF, HTML, ODT, or even DOC files. This will be useful for easy document creation, hopefully even with limited knowledge of the document language!

Test One

The first test will be to convert from a markdown text file (this file) into html. Eventually we will see how many complex layouts and interops we can do, but for now we just want to display a simple webpage-like document. It will be read in a browser and is therefore incredibly portable!

If I want to include a code snippet I can use code blocks:

myVector <- c("A","B","C")
myDF <- rnorm(10,3,5)

This will make the document a little better looking and more readable. I'll have to investigate the pandoc documentation to see if I can automatically fit that in when converting the document!

Update on auto styling

I was able to include styling in my document by writing the html I wanted to include in a new document, which I named styles.css.

Then I called pandoc using the -H flag, with the styles.css file as the argument. The full command looked like this: pandoc pandocs_test.md -s -H styles.css -o testoutput.html
